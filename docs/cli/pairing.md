# `dryads-ai pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
dryads-ai pairing list whatsapp
dryads-ai pairing approve whatsapp <code> --notify
```
