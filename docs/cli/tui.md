# `dmms-ai tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
dmms-ai tui
dmms-ai tui --url ws://127.0.0.1:18789 --token <token>
dmms-ai tui --session main --deliver
```
